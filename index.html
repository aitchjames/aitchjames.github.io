Visit My Journal

After completing this web application that allows users to sign up and post journal entries. I believe I have gained the skills needed for future web applications within the Node/JavaScript ecosystem. This web application uses sessions and cookies to authenticate users within the web browser.

The journal entries users post can use Markdown in their posts to allow them use bold and italic text and create headings within their entries.

This web application also features a public chat room to allow users to chat with eachother within the app, this was achieved with socket.io. To allow bi-directional communication from the client and the web server.
