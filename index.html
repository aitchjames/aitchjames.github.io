GuardIT Solutions

WordPress WooCommerce Custom Theme Responsive UI JavaScript Mailchimp SiteGround

I was recommended to GuardIT Solutions from a designer friend to fix their website and design. As the main issues with the website were the design not being responsive to each screen size, critical product pages not being developed towards the PSD design, as well as slow load speeds and rendering.

After digging through the WordPress backend, I noticed there was a lot of bloated plugins being used to accomplish small tasks. I decided to create a fresh custom theme from scratch, only using necessary plugins such as WooCommerce and Advanced Custom Fields.

After developing the new website for GuardIT Solutions, there were noticeable improvements such as faster loading times from 7 seconds to 1 second with rendering. Network requests reduced by two-thirds, this was due to fewer plugins used.

Zesty Recipes

Responsive UI JavaScript Node.js Express MongoDB Digital Ocean

Zesty Recipes is a passion project I've been working on in my spare time. It's a well-designed food recipe web application developed and designed by me.

I created and designed the brand that is simple enough to be rememberable but also allow it's content to do the talking. So the users will want to find new and exciting recipes to try.

To create this web application, I wanted to use the Node.js ecosystem and to expand on my knowledge and skills with JavaScript. I knew I wanted to make this web application server-side rendered to allow it to be SEO friendly, so users can easily find recipes as search engines have an easier time indexing the web pages.

The challenge of the brief was to create new illustrations of the cocktails for the menu and while keeping the brands' integrity. I photographed the cocktails for reference for its true colours, glass shape and size.
